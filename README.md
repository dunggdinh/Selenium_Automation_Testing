# Selenium Automation Testing (Java)

Dự án Java + Selenium với ít nhất 03 test case tự động cho website mẫu `https://www.saucedemo.com/`.

## Chạy test trên bash

1. Cài Java 17 và Maven.
2. Cấp quyền thực thi và chạy script:
   ```bash
   chmod +x run-tests.sh
   ./run-tests.sh
   ```

## Cấu trúc dự án

- `pom.xml`: cấu hình Maven và dependencies
- `src/test/java`: chứa test Selenium
